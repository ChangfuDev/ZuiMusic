# ZuiMusic音乐播放器

界面模仿网易云音乐（旧版），数据来自于百度音乐api。<br>

仅用做学习与交流。<br>

大致是采用MVP架构，网络请求使用Retrofit2.0+RxJava，图片加载使用Glide。<br>
